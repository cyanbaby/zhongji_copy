# mv命令
目标是移动 node_modules

[参考](https://zhidao.baidu.com/question/2018527220749468108.html)

```shell
zhangg@DESKTOP-IHPUCAV MINGW64 ~/demos/demo/linux-shell-test/test-mv
$ mkdir temp01

zhangg@DESKTOP-IHPUCAV MINGW64 ~/demos/demo/linux-shell-test/test-mv
$ mkdir temp02

zhangg@DESKTOP-IHPUCAV MINGW64 ~/demos/demo/linux-shell-test/test-mv
$ mv temp01 temp02
# 将 temp01 移动到 temp02 里

zhangg@DESKTOP-IHPUCAV MINGW64 ~/demos/demo/linux-shell-test/test-mv
$ mv temp02/temp01/ ./
# 将 temp02/temp01/ 移动到 当前目录里



```
