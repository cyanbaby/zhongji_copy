## How to learn ?
- 不限于掘金小册，极客学院，等等专题课程（学什么知识模块例如：webpack）目录很重要，用于其他众筹到的视频课查漏补缺。
- 课程很多，要学会总结出东西，淘出金子。
- 大模块要拆分去学，要有目的性去学。例如：前端数据结构与算法，分为数据结构和算法去学，最终扩展到项目中去用，面试题的理解。
- 知识树（knowledge tree） 可以先从平铺的.md文件和平铺目录开始，后续在分类。
- 5w学习法!
- 用 google 搜索!
